# Очистка изображений по колиброванным данным

Утилита с графическим интерфейсом для предварительной постобработки кадров в задачах компьютерного зрения и вычислительной фотографии. Программа предназначена для калибровки изображений путем устранения аппаратного шума сенсора и оптических искажений.

## Возможности
* **Генерация Master-кадров:** Автоматическое математическое усреднениетемновых и плоских (белых) кадров для создания эталонных масок дефектов.
* **Очистка целевого изображения:** Точное вычитание темнового шума и нормализация освещенности тестового кадра с защитой от потери данных.
* **GUI на Tkinter:** Интерфейс с выводом исходного и скорректированного изображений.
